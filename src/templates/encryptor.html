▌ COLDPHRASE · builder
offline · single-file · argon2id — cold storage for crypto seed phrases

Encrypt a recovery phrase

Enter your phrase Trust Wallet-style; you get one self-contained HTML file that decrypts fully offline.

  • Argon2id (256 MiB, 4 passes) → HKDF-SHA512 → AES-256-GCM with authenticated parameters
  • Duress mode: a second password reveals a decoy you choose, not the real secret
  • Runtime library hash check + network-blocking CSP
Add duress mode (optional)

If forced to unlock, hand over this second password to reveal the decoy instead of the real secret. To be believable, make the decoy a real low-value wallet — not an empty one.

🐍 Disguise output as a Snake game

The file opens as a playable Snake game. It reveals the unlock screen only after someone types your secret word (any casing). Only its SHA-256 is stored, not the word.

The password is unrecoverable — if you forget it, the content is gone forever.
Password strength is the whole ballgame. Use “generate strong”; the meter is only a rough guide and over-rates patterned passwords.
Never keep this file or the original text in a cloud-synced folder (OneDrive, Windows Documents, …), or the offline-attack target is copied to a third-party server.
After building, test the file with every password, then delete the original and empty the recycle bin.
